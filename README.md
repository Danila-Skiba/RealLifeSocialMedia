### Инструкция по запуску:

1. Создайте виртуальное окружение 

```
    python3 -m venv venv
```
2. Активируйте 

```
    source venv/bin/activate
```

3. Установите зависимости 
`pip install -r requirements.txt`

4. Установите Caddy

5. Запустите скрипт `start_all_with_proxy.sh`

6. Перейдите на `http://localhost:8080`

#### Ветка feature/websocket-service

1. Создать WebSocket сервер в папке `server`

2. Реализовать отправку сообщений с интервалами 2-5 секунд:
    - 65% вероятность: «Это лучшая тема, что есть. Хотя…»
    - 35% вероятность: «Надо менять тему…»

3. Добавить обработку уникального сообщения от клиента

4. Научить сервер отвечать «Да? Хорошо. Спасибо за поддержку!» на уникальное сообщение

5. Тестирование сервиса в файле scripts/test_websocket.py

#### Ветка feature/client-server

1. Создание сервер-клиента в папке client

2. Сервер возвращает страницу с загаловком h1 «Была выбрана тема …»

3. Добавить пустой h2, пустой p и кнопку "Отправить сообщение"

4. Подключить js скрипт к странице

5. В js скрипте создать сокет-соединение с сервером

6. При успешном handshake записать информацию в h2

7. Создать слушателя событий:

    - Очищать p при получении события

    - Вставлять в p полученное от сервера сообщение

8. Активировать кнопку только при успешном соединении 

9. При нажатии на кнопку отправлять уникальное сообщение

4. Написать .bat файл для запуска всех трёх серверов
