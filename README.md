## Запуск сервера

1. Добавляете файлик .env в директорию theme-project/

2. Запускаете сервер: `docker-compose up -d --build`, перейдя заранее в папку theme-project/

Приложение будет работать, если запускаете клиент OmSTUSchedule (на Flutter) и сервер на одной машине (данные передаются по localhost). 

_На физическом устройстве работать не будет, а на эмуляторе да_
